# PRPA: Post-Render Perspective Align

## Install

Install from pypi:
```sh
pip install prpa
```
or from source:
```sh
pip install git+https://github.com/yindaheng98/PostRenderPerspectiveAlign
```

## Dependencies
- PyTorch  
- NumPy  
- open3d (optional, for visualization)
- opencv (optional, for reading sample data)
